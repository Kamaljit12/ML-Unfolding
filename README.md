## `2. FOUNDATION (VERY IMPORTANT)`

Before deep ML, master:

• Linear Algebra
  - vectors, matrices, eigenvalues, SVD

• Calculus
  - derivatives, partial derivatives, gradients
  - chain rule (VERY IMPORTANT for backprop)

• Probability & Statistics
  - distributions (Gaussian, Bernoulli)
  - expectation, variance
  - Bayes theorem

Without this → ML will feel like magic (and you don’t want that).

## `3. UNFOLD ALGORITHMS (CORE APPROACH)`

For every algorithm, follow this pattern:

STEP 1: Problem it solves
STEP 2: Mathematical formulation
STEP 3: Optimization objective
STEP 4: How parameters are learned
STEP 5: Limitations
STEP 6: Improvements / variants

Example (Linear Regression):

1. Problem:
   Predict continuous values

2. Equation:
   y = wx + b

3. Loss:
   L = (y - ŷ)^2

4. Optimization:
   Gradient Descent

5. Insight:
   It minimizes squared error → assumes linear relationship

6. Limitation:
   Cannot model non-linear patterns

This is called **first-principles understanding**.

## `4. BUILD FROM SCRATCH (VERY IMPORTANT)`

Do NOT rely only on sklearn or TensorFlow.

You must implement:
• Linear Regression (from scratch)

## `5. THINK LIKE A RESEARCHER`

• What if features are high-dimensional?

• Can I reduce overfitting?
• Can I change the loss function?

This is how research papers are born.
